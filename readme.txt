== Installation ==

1. Upload the entire `ucam-staff-lookup` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu within the WordPress Dashboard.

You will now see the `UCAM Lookup` menu in your WordPress admin panel.
